# Hippo fetch 
System Fetch in C++ 

![alt text](https://github.com/w8ste/Hippofetch/blob/main/.images/terminal_transparent.png)

### Supported Package Managers
This is needed for the package count.
- Pacman (*tested*)
- Apt-get (*not tested*)
- Yum (*not tested*)
- Emerge (*not tested*)
- Apk (*not tested*)
- Zypp (*not tested*)
